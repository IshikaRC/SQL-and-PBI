# SQL-and-PBI
**Promotion Performance Analytics Dashboard**

**Overview**
This project analyzes the effectiveness of promotional campaigns using SQL and Power BI. The dashboard evaluates customer participation, sales impact, profitability, and promotion performance across markets and customer segments.

**Tools & Technologies**
SQL (MySQL)
Power BI
DAX
Excel

**Dataset**
The analysis was performed using 4 datasets:
Customers
Orders
Promotions
Promotion Participants

**Key Analysis Performed**

Qualification Rate Analysis
Sales Before vs During Promotion
Sales Lift % Analysis
Promotion Profitability Analysis
Customer Participation Analysis
Repeat Purchase Analysis
Sponsor-Level Performance
